## Zig Lox Port

This is a port of the Lox Programming language. This is my attempt of porting the Lox Compiler by Robert Nystrom in the book Crafting Interpreters 

## Project description

This port is a way from me to learn compilers and Zig at the same time. It is a slow burn and will stop at Ch 17 for a while before moving on to other language features.

## Who this project is for

This project is intended for me who wants to learn Zig and how compilers work.
